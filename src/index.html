Hind-cite Scraper

This houses the scraper and scraper monitor code.

Note - the script code is in /opt/hind-cite-scraper

IMPORTANT: you need to put a proper credentials file in /opt/hind-cite-credentials.json. You have to do this manually. So if you destroy your vagrant setup, you have to do it again. (There are more elegant ways to do this, but I just want to get this running again.)

For more information check the Github repo.
